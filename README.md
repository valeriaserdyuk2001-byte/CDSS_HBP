# AH rule-based prototype

Тестовый прототип сайта для системы помощи принятия решений в выборе терапии артериальной гипертензии.

## Что внутри
- `index.html` — страница ввода данных
- `result.html` — страница вывода диагноза
- `styles.css` — стили
- `script.js` — логика первой страницы
- `result.js` — вывод на второй странице

## Как запустить
Открой `index.html` в браузере.
